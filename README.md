# I-310D-Project-AAC-Dog-Adoption-ML

In this project, we trained and evaluated ML models to predict whether a given dog will be adopted at the Austin Animal Center (AAC). This repository contains the code we wrote and used to do so. A full report of this project is available at https://medium.com/@jaquelineochoa/how-we-used-machine-learning-to-analyze-austin-animal-center-dog-adoptions-7b20dddd0c53, and the raw data we used, as well as the cleaned data and models resulting from them, are  available at https://data.world/zhao993059/i-310d-project-aac-dog-adoption-data.
